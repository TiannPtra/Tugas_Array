## 1. Penjelasan Array
Array adalah sekumpulan data yang menyimpan banyak data dengan tipe yang sama dalam satu variabel dan disusun secara berurutan serta tiap data datanya memiliki indeks supaya mudah. Pada Python, array disebut list, pada tugas 01 kali ini kita menggunakan array untuk mencari nilai tertinggi dan terendah. Dengan demikian kita menggunakan array lebih mudah. Array memiliki indeks dan dimulai dari 0, sehingga kita menghitung data secara berurutan dan sederhana.
